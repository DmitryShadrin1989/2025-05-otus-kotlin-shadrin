## Группа: Kotlin-2025-05

### Система для создания, редактирования и хранения личных заметок с напоминаниями и категоризацией. Основной интерфейс — Telegram‑бот.

## Ключевые возможности

- регистрация и аутентификация пользователей;
- создание, редактирование и мягкое удаление записей;
- установка напоминаний (push через Telegram);
- поиск и фильтрация записей по заголовку, тексту и категориям;
- управление категориями (создание, редактирование, удаление).

## Архитектурные решения

### СУБД

**PostgreSQL** — выбрана как основная база данных:
- гарантирует целостность данных (ACID);
- поддерживает сложные запросы;
- масштабируема и соответствует промышленным стандартам.

> Для локальной разработки используется Docker.

### Интерфейс пользователя

**Telegram‑бот** — основной UI для MVP:
- мгновенный доступ без установки ПО;
- экономия времени на разработку;
- поддержка базовых функций через Telegram Bot API.

> В перспективе — добавление веб‑ и мобильного интерфейсов.

### Основные компоненты

- **Бэкенд**: Kotlin/Spring — обрабатывает бизнес‑логику.
- **БД**: PostgreSQL — хранит все данные.
- **Интерфейс**: Telegram Bot API — взаимодействие с пользователем.
- **Инструментарий**: Docker — локальное развёртывание.

## Взаимодействие компонентов

1. Пользователь отправляет команду через Telegram‑бота.
2. Бот передаёт запрос на бэкенд.
3. Бэкенд обрабатывает запрос и взаимодействует с PostgreSQL.
4. Результат возвращается боту и отображается пользователю.
